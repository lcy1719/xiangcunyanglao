【系统功能模块】
本系统包含管理员、老人、医疗人员、乡村志愿者4个角色。
管理员角色：
1. 用户管理：管理系统中所有用户的信息，包括添加、删除和修改用户。
2. 生活照料管理：添加、删除、修改和查看生活照料信息。
3. 医疗信息管理：添加、删除、修改和查看医疗信息。
4. 医疗类型管理：添加、删除、修改和查看医疗类型信息。
5. 系统配置管理：管理系统配置参数，如上传图片的路径等。
6. 权限管理：分配和管理不同角色的权限。
老人角色：
1. 个人信息管理：老人可以管理自己的个人信息，包括姓名、联系方式、邮箱等。
2. 查看生活照料信息：老人可以查看生活照料信息。
3. 查看医疗信息：老人可以查看医疗信息。
医疗人员角色：
1. 个人信息管理：医疗人员可以管理自己的个人信息，包括姓名、联系方式、邮箱等。
2. 医疗保健管理：医疗人员可以添加、删除、修改和查看医疗保健信息。
3. 健康档案管理：医疗人员可以添加、删除、修改和查看健康档案信息。
乡村志愿者角色：
1. 个人信息管理：乡村志愿者可以管理自己的个人信息，包括姓名、联系方式、邮箱等。
2. 活动报名管理：乡村志愿者可以报名参加文娱活动，并查看报名状态。
3. 土地承包管理：乡村志愿者可以承包土地，并查看承包信息。
